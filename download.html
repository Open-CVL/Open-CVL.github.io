Dataset Access

Select OpenCVL release archives.

The full OpenCVL dataset is distributed as multiple tar archives, so you can download only the source or country split you need. ZOD archives are publicly available now; the remaining release archives are currently being uploaded to the data-sharing platform.

How to Use

Download archives into one folder, then extract them together.

Each archive expands into the same OpenCVL/ root. Download the selected archives, keep the corresponding .sha256 files beside them, and verify each file before extraction. 

sha256sum -c *.sha256
for f in OpenCVL_*.tar; do
  tar -xf "$f"
done
